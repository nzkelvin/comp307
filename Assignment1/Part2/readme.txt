==Steps==
* Open a Windows command line Console.
* Navigate to the directory called "Part2"
* Please run the follow command in the Windows console to see the accuracy output and the decision tree structure from 'hepatitis-training.dat' and 'hepatitis-test.dat':
python .\DecisionTreeMain.py
* Please run the follow command in the Windows console to see the accuracy outputs from 10 pairs of training/test files:
python .\DecisionTreeMain.py 0 10

==Parameters==
* Parameter 1 = Use 0 for hiding decision tree, and use 1 for printing decision trees.
* Parameter 2 = Number of training and test runs. You should put an integer number between 1 to 10(inclusive).
Any number outside the range will cause the programme to crash.
